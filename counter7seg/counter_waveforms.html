Sample behavioral waveforms for design file counter.vhd

The following waveforms show the behavior of lpm_counter megafunction for the chosen set of parameters in design counter.vhd. The design counter.vhd is a 4 bit dynamically controlled up/down counter. The counter has an asynchronous set signal, that sets the counter to all 1's. The counter has an asynchronous clear input.
